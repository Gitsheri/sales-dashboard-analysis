# Sales Performance Dashboard

## Project Overview
This project analyses retail sales data to identify trends in revenue, profit, category performance, top products, and regional sales. The final output is an interactive Power BI dashboard designed to support business decision-making.

## Objectives
- Analyse sales trends over time
- Identify top-performing regions and categories
- Compare sales and profit performance
- Highlight top-performing products
- Provide business recommendations based on the analysis

## Tools Used
- Excel
- Power BI
- GitHub

## Dataset
The dataset contains retail transaction data including order date, sales, profit, quantity, category, sub-category, product name, region, and customer segment.

## Dashboard Features
- KPI cards for Total Sales, Total Profit, Total Orders, and Average Order Value
- Sales over time
- Sales by region
- Sales by category
- Profit by category
- Top 10 products by sales
- Region slicer for filtering

## Key Insights
- West recorded the highest sales among all regions.
- Technology was the top-performing category in both sales and profit.
- Furniture generated solid sales but lower profit compared to Technology.
- A small number of products contributed a large share of total sales.
- Total sales reached $2.33M, with total profit of $292.3K across 5,111 orders.
